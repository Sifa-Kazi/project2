Project:- Customer Churn Analysis for Telecom Industry

Objective

To predict customer churn and derive actionable strategies to retain users in a competitive telecom environment.

Dataset Used

Telecom Customer Churn Dataset (Kaggle: churn-bigml-80.csv)

Steps Performed:-

•	Loaded dataset and explored first 10 records

•	Converted churn labels to binary format

•	Encoded categorical variables (International plan, Voice mail plan)

•	Stored dataset in SQLite for aggregation and analysis

•	Performed SQL-based analysis on customer complaints and call usage

•	Conducted EDA using pie charts, boxplots, and line charts

•	Trained a Random Forest Classifier for churn prediction

•	Evaluated model with confusion matrix and classification report

•	Used SHAP for model explainability and feature importance

•	Segmented customers into At Risk, Loyal, and Dormant

•	Created visual insights for churn distribution, complaints, and tenure trends

Tools Used

•	Python (Pandas, NumPy, Matplotlib, Seaborn)

•	Scikit-learn

•	SQLite

•	SHAP

•	Jupyter Notebook

Output

•	PowerPoint presentation with visual insights, customer segmentation, and actionable final recommendations
